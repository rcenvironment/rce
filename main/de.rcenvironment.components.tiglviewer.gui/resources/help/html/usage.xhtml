TiGL Viewer Component Usage

Synopsis

The TiGL Viewer component shows CPACS geometries during workflow execution.

Note: The TiGL Viewer Component is currently only available on Windows operating systems.

Usage

The TiGL Viewer component has no properties at all. It only has a preconfigured input channel “TiGL Viewer File”.

During workflow execution the view “TIGL Viewer” will be opened during the first component run. On following iterations the geometry shown in the view will be updated with the current input.
